      示例 #

      单点登录演示 #

      现在有两个应用测试应用1和测试应用2

      登录 #

      测试应用1 测试应用2

      点击其中一个应用, 要求登录, 输入admin/admin实完成登录, OAuth2NSSO会回掉返回code, 使用code该应用可以调用 接口1-2 获取access_token.

      再次点击另外一个应用, 无需登录, 直接回掉返回code. 同样, 使用code该应用可以调用 接口1-2 获取access_token.

      退出 #

      在测试应用1或测试应用2中都可以调用退出逻辑

      退出

      点击退出按钮, 会重新回到登录页面, 该服务退出后, 再次点击测试应用1/测试应用2会要求登录.
